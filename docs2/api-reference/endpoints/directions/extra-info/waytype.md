# WayType

Value list for the directions response values in

```jsonpath
$.routes[*].extras.waytype.values
```

This extra provides info about the type of the way of the corresponding parts of the route.

| Value |     Name     |
|:-----:|:------------:|
| 0     | Unknown      |
| 1     | State Road   |
| 2     | Road         |
| 3     | Street       |
| 4     | Path         |
| 5     | Track        |
| 6     | Cycleway     |
| 7     | Footway      |
| 8     | Steps        |
| 9     | Ferry        |
| 10    | Construction |
