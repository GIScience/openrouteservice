# Instruction Types
The following table contains the encoding of the instruction types present in the `type`-field in a step of a segment of a route.

| Value |     Encoding     |
|:-----:|:----------------:|
| 0     | Left             |
| 1     | Right            |
| 2     | Sharp left       |
| 3     | Sharp right      |
| 4     | Slight left      |
| 5     | Slight right     |
| 6     | Straight         |
| 7     | Enter roundabout |
| 8     | Exit roundabout  |
| 9     | U-turn           |
| 10    | Goal             |
| 11    | Depart           |
| 12    | Keep left        |
| 13    | Keep right       |
